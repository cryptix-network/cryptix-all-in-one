# Software Overview

This software utilizes pre-built executable applications, which can be replaced (self-built) if desired. The Node uses the Rusty Cryptix version, while the Wallet uses the Web Wallet. The Miner supports both GPU and CPU. All modules can be started individually.

## FAQ

### Is the new V3 version compatible with previous versions?

Yes. Older versions can be easily overwritten by the new V3 release. No complex migration or setup is required.

### What are the system requirements?

The software is very lightweight:

- RAM: about 50 MB
- CPU: about 1% of a single CPU core
- Disk space: around 500 MB

It is also portable, meaning no traditional installation is needed.

### Do I need special hardware for mining?

No special hardware is required.

- You can mine using a CPU, even a single-core processor
- GPU mining is also supported for better performance

### Does performance depend on hardware?

Yes. While mining is possible on almost any system, better hardware, especially GPUs, will result in higher efficiency and performance.

### Can I run a node and mine at the same time?

Yes, but keep in mind that running a node and mining will require additional system resources depending on your usage.
